# Cursor Website Clone

A high-fidelity, recreation of the [Cursor.com](https://cursor.com/) landing page, built using pure HTML and CSS. This project focuses on replicating the modern, dark-themed aesthetic, responsive layout, and typography of the AI code editor's official website.
